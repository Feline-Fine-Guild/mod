# Feline Finecraft

For version 1.21.1.

A custom mod for Feline Finecraft to add custom items and blocks 
to the Minecraft server.<br>
* See [CONTRIBUTING.txt](https://github.com/Feline-Fine-Guild/mod/blob/master/README.txt)
to learn how to contribute to the project.
* Visit https://feline-fine-guild.github.io for more information (redirects to Discord).

Feature(s):
* Adds Feline Fine logo as an item
